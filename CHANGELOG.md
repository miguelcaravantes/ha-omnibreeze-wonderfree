# Changelog

## v1.0.0 — 2026-07-15

- First public release.
- Added local LAN control for the Wonderfree-based OmniBreeze DC2313R tower fan.
- Added power, five speeds, oscillation, Normal/Natural/Sleep/Automatic presets, temperature, Sound, LED Display, and a 1–12 hour auto-off timer.
- Added one-time Wonderfree account authentication for local-key retrieval across Europe/Latin America, North America, and China.
- Added HACS custom-repository support and English documentation.
